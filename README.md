## 프로젝트 소개

'**the survey**'는 설문조사를 생성하고 참여할 수 있는 플랫폼입니다.

**무분별한 설문조사 게시를 방지**하기 위해, 게시를 위해서는 일정 포인트가 필요하며, <u>**설문조사 참여에 대한 보상**</u>으로 포인트를 얻을 수 있습니다.
